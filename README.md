
![](https://i.imgur.com/WCfMsYA.gif)

Require :
---------
- make

Installation : 
--------------
```
git clone https://github.com/yanuarakhid/gicom.git
```
```
cd gicom && sudo make install
```

Run it in the folder that you want to commit or push

```
gicom
```

Uninstall:
----------
```
sudo make uninstall
```
